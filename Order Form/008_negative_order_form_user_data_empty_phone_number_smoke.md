* Предусловие:
 
  1.Пользователь не авторизован
 
  2.В корзину добавлен один товар
 
  3.Открыта "Корзина" с товаром
 
  4.Нажать на кнопку "Перейти к оформлению заказа"
 
  5.Открыта форма "Оформление заказа" - "Доставка и оплата" шаг 1
  
  6.Способ доставки "Самовывоз"
  
  7.Открыто окно "Шаг 2" - ввод данных для физ. лица
 

* Шаги:

  1.Поле "Номер телефона" оставить пустым
  
  2.Все остальные поля заполнены валидными данными
  
  3.Выбрать способ оплаты "Оплата на сайте"
  
  4.Нажать на кнопку "Оформить заказ"
 

* Ожидаемый результат:

   Поле обводится красной рамкой и под ним появляется текст красным цветом "Укажите номер телефона в формате +79хх-ххх-хх-хх"
