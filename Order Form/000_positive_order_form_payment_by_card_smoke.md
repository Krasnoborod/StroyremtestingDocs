 Форма оформление заказа, самовывоз с оплатой на сайте
 
 
 тест сервер - https://test2.stroyrem-nn.ru/
 
 прод сервер - https://stroyrem-nn.ru/
 
* Предусловие:

  1.Пользователь не авторизован

  2.В корзину добавлен товар
  
  3.Открыта "Корзина"

* Шаги:

  1.Нажать на кнопку "Перейти к оформлению"
  
  * ОР: Открывается "Шаг 1" - окно для выбора способа доставки и оплаты товара
  
  2. Выбрать "Самовывоз"
  
  * ОР: Открывается окно с календарем для выбора даты, график работы магазина, выбор способа оплаты и сумма для оплаты 
  
  3. Выбрать дату в календаре, способ оплаты "Другой способ оплаты", отметить в чекбоксе согласием на обработку личных данных (неизвестно это будет или нет) и нажать на кнопку "Продолжить"
  
  * ОР: Открывается окно "Шаг 2" с полями для заноления информации Физ.лица
  
  4. Поля ФИО, номер телефона, электронная почта заполнены валидными данными, способ оплаты выбрать "Оплата на сайте" нажать на кнопку "Оформить заказ"
  
  *ОР: Открывается шлюз Сбербанка
  
  5. Все поля шлюза заполнены валидными данными, нажать на кнопку "Оплатить"
  
  * ОР: Открывается окно с информацией об успешной оплате заказа, информация о самом заказе. На почту приходить подробная информация о пункте самовывоза и информацией о заказе.
